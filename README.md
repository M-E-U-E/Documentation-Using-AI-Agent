# Set Up Virtual Environment
   **Create virtual environment On macOS/Linux:**
```
python3 -m venv env
source env/bin/activate
```
**Create virtual environment On Windows:**
```
python -m venv env
venv\Scripts\activate
```
**Install Dependencies**
```
pip install -r requirements.txt
```
